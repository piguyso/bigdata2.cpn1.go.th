EE.CPN1 UX/UI REDESIGN

แผนภาพและตัวอย่างระบบต้นแบบตามโครงสร้าง e.cpn1.go.th

Strategic Plan

แนวทางการยกระดับ UX/UI และทิศทางการดีไซน์

การปรับโครงสร้างหน้าเว็บหลักของ ee.cpn1.go.th ให้มีเอกลักษณ์ร่วมและดูหรูหราแบบเดียวกับ e/e.cpn1.go.th โดยใช้การออกแบบสไตล์ **Dark-Slate Mode, Glassmorphism** และใช้โทนสีลุ่มลึกอย่างสีคราม (Indigo) และเขียวมรกต (Emerald) เพื่อมอบมิติที่ทันสมัย น่าใช้งาน และมีระบบตอบสนองที่ไหลลื่น (SPA-like feeling)

1. ระบบโทนสีและการไล่เฉด (Color Palette)

  • ใช้ Slate 900 / 950 เป็นสีพื้นหลังหลักลดความกระด้าง
  • เฉดสีกึ่งโปร่งใส (Glassmorphism) บนการ์ดและปุ่ม
  • ใช้สี Indigo-600 นำทางหลัก และสี Emerald-500 สื่อถึงความสำเร็จ/อนุมัติ

2. ระบบเมนูและเลย์เอาต์ (Translucent Nav)

  • เปลี่ยนแถบเมนูด้านบนเป็นแบบโปร่งแสง ลอยตัว (Floating Glassbar)
  • ใช้การเบลอฉากหลังเพื่อรักษาความสะอาดตา (`backdrop-blur-md`)
  • เมนูสัมผัสและปุ่มดำเนินการมีขนาดใหญ่ขึ้น (Min. 44x44px สำหรับ Touch Target)

3. ความปลอดภัยและฟอร์มรูปแบบใหม่ (Safe Inputs)

  • ฟอร์มเข้าสู่ระบบและแก้ไขข้อมูลรองรับความปลอดภัยตามหลัก OWASP
  • ฟิลด์รหัสผ่านมีระบบสลับประเภทแสดงผล (Show/Hide Toggle)
  • แถบประเมินความปลอดภัยของรหัสผ่านแบบเรียลไทม์ (Dynamic Strength Bar)

ขั้นตอนการเปลี่ยนผ่าน UI (Implementation Roadmap)

1

การตั้งค่าตัวแปรการออกแบบ (Design Tokens Config)

อัปเดตไฟล์ `tailwind.config.js` ในระบบหลักเพื่อตั้งค่าสีพื้นฐาน `slate-950`, `slate-850`, เอฟเฟกต์เบลอ และเพิ่มฟอนต์ Anuphan + Sarabun สำหรับการนำเสนอภาษาไทยอย่างมีเอกลักษณ์

2

อัปเกรดเลย์เอาต์หน้ากากครอบ (Base Layout Integration)

แทนที่โครงสร้าง `` เดิมด้วยธีมสีทึบ/โปร่งแสง และจัดการระบบล็อกอิน/เมนูของ Admin ให้มี Dropdown คล้ายกับระบบ E-Portal ดึงประวัติและไอคอนผู้ใช้มาสร้างเป็น Badge ที่มุมขวา

3

ปรับปรุงแบบฟอร์มและการโต้ตอบ (SPA Form UX)

นำรหัสโครงสร้างปุ่ม Loading (`fa-circle-notch fa-spin`) ตัวแจ้งเตือนด่วน (Toast Alert) และระบบอัปโหลดรูปภาพผ่าน Cropper.js 16:9 ไปใช้ร่วมกันในส่วนบริหารจัดการของ LMS และข้อมูลเครือข่ายโรงเรียน

ระบบสาธิตส่วนติดต่อผู้ใช้งานจริง (Live Sandbox Prototype)

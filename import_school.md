# แผนการนำเข้าข้อมูลรายชื่อโรงเรียน (BOPP DMC)

ระบบได้ทำการย้ายไปใช้แนวทาง **Local Database Cache** เพื่อให้สามารถทำงานได้รวดเร็วขึ้น ไม่มีปัญหาระบบค้าง (Timeout) หรือหน่วยความจำเต็ม (Memory Exhaustion) จากไฟล์ขนาดใหญ่ของประเทศ (28,841 โรงเรียน) และสามารถติดตั้งแบบ Offline ได้ง่ายผ่าน Seeder

---

## 🛠 ข้อมูลสถาปัตยกรรมระบบ

1. **ตาราง `dmc_schools` (Cache Table)**
   - เก็บข้อมูลดิบของโรงเรียนทั้งหมด 28,841 โรงเรียนจากไฟล์ประเทศ
   - เพิ่ม `Index` ในคอลัมน์สำคัญ ได้แก่ `areacode`, `smis`, `ministry`, `area_name` เพื่อการค้นหาที่รวดเร็ว

2. **ระบบกู้คืนข้อมูล / ติดตั้งเครื่องใหม่ (Seeder)**
   - **`DmcSchoolSeeder`** จะอ่านไฟล์ `DMC691.xlsx` จาก root ของโปรเจกต์โดยใช้ `XMLReader` แบบสตรีม และทำ Batch Insert ทีละ 500 แถว
   - ใช้เวลาในการ Seed ทั้งประเทศเพียงประมาณ **10-15 วินาที** (ไม่มีการจองแรมขนาดใหญ่)

3. **การนำเข้าเข้าตารางจริง (`system_school`)**
   - เมื่อกดปุ่มในหน้าควบคุม ระบบจะทำการกรองเฉพาะโรงเรียนของเขตพื้นที่ปัจจุบัน (เช่น ชัยภูมิ เขต 2) โดยใช้รหัสเขตพื้นที่ (Areacode) เป็นหลักในการ Match
   - มีระบบ fallback ไปหาชื่อเขตภาษาไทยหากหาตัวเลขไม่พบ
   - รองรับ 2 โหมด: **Replace** (ลบและเขียนทับเฉพาะโรงเรียนในเขตนี้) และ **Merge** (เพิ่ม/อัปเดต โดยอิงตามรหัส SMIS)

## 🚀 ขั้นตอนการติดตั้งและการนำเข้าข้อมูล

### 1. การเตรียมไฟล์ดิบ (สำหรับเครื่องติดตั้งใหม่)
นำไฟล์ **`DMC691.xlsx`** ไปวางไว้ที่ root ของโปรเจกต์ (โฟลเดอร์ `d:\bigdata\DMC691.xlsx`)

### 2. รันคำสั่งสร้างตารางและนำเข้า Cache
```bash
# รัน migration เพื่อสร้างตารางเก็บ cache และ logs
php artisan migrate

# รัน seeder เพื่อโหลดข้อมูลจากไฟล์เข้า cache table
php artisan db:seed --class=DmcSchoolSeeder
```

### 3. ตรวจสอบและนำเข้าผ่านหน้าเว็บ (UI)
1. ไปที่เมนู **นำเข้าข้อมูล -> รายชื่อโรงเรียน (DMC)** (หรือเข้าไปที่ `/admin/school-import`)
2. ระบบจะขึ้นจำนวนโรงเรียนที่รอบรรจุในตาราง Cache (เช่น 28,841 แห่ง และจำนวนที่ตรงกับเขตพื้นที่นี้ 258 แห่ง)
3. เลือกโหมดที่ต้องการ: **Replace** หรือ **Merge**
4. กดปุ่ม **"ตรวจสอบก่อนนำเข้า"** เพื่อ Preview ดูรายชื่อ 15 โรงเรียนแรกของเขตท่าน
5. เมื่อมั่นใจแล้ว ให้กด **"ยืนยันนำเข้าข้อมูล"**
